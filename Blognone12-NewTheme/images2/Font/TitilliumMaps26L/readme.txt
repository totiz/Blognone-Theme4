How to install this font on your computer?

For Windows 7 / Vista users:

1. Unzip the downloaded archive.
2. Right-click the font file(s) and choose "Install".

For users of the previous Windows versions:

1. Unzip the downloaded archive.
2. Copy the included file(s) into a default Windows font folder 
   (usually C:\WINDOWS\FONTS or C:\WINNT\FONTS)

For Mac users:

Mac OS X 10.3 or above (including the FontBook)

1. Unzip the downloaded archive (you may use Stuffit Expander, 
   Zipeg, or similar utility)
2. Double-click the font file and hit "Install font" button at 
   the bottom of the preview.

Mac OS X

1. Unzip the downloaded archive (you may use Stuffit Expander, 
   Zipeg, or similar utility)
2. Either copy the font file(s) to /Library/Fonts (for all users), 
   or to /Users/Your_username/Library/Fonts (for you only).

Mac OS 9 or earlier

1. Unzip the downloaded archive (you may use Stuffit Expander, 
   Zipeg, or similar utility)
2. You have to convert the font file(s) you have downloaded. 
   Drag the font suitcases into the System folder. The system 
   will propose you to add them to the Fonts folder.

For Linux users:

1. Unzip the downloaded archive (use 7zip or similar software).
2. Copy the font file(s) to /USR/SHARE/FONTS
